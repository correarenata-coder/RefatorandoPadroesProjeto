## 📖 Livro 

- **📌 Título:** *Refatorando com Padrões de Projeto: Um Guia em Java*  
- **🎯 Tema:** Melhoria de código existente utilizando padrões de projeto  
- **👨‍💻 Autor:** Kerievsky, Joshua  

> ⚠️ **Observação:** Embora o livro utilize **Java** nos exemplos, neste repositório todas as implementações e exemplos práticos serão desenvolvidos em **C#**.

---

## 💡 Sobre o Livro

Este livro mostra como **refatorar código legado** de forma segura, aplicando **padrões de projeto** de maneira incremental.

O foco não é apenas teoria, mas **como evoluir o código na prática**, tornando-o mais limpo, flexível e sustentável.

---

## 🧠 Tópicos abordados neste livro

- Refatoração de código legado  
- Aplicação prática de **Design Patterns**  
- Substituição de condicionais complexas  
- Introdução gradual de padrões  
- Melhoria de legibilidade e manutenção  
- Boas práticas em Java  

---

## 🚀 Objetivo da leitura

- Aprender a melhorar código existente sem quebrar funcionalidades  
- Entender quando e como aplicar padrões de projeto  
- Reduzir complexidade e acoplamento  
- Escrever código mais limpo e sustentável  

---

## 📌 Exemplos de padrões abordados

- Strategy  
- Factory Method  
- Observer  
- Template Method  
- Decorator  

---

## 🧩 Como aplicar no dia a dia

1. Identifique trechos de código difíceis de manter  
2. Refatore aos poucos (pequenos passos)  
3. Introduza padrões quando fizer sentido  
4. Garanta cobertura de testes antes das mudanças  
5. Valide melhorias na legibilidade e manutenção
